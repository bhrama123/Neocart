# 🛒 NeoCart - MERN E-Commerce Website

## 📌 Project Description

NeoCart is a Full Stack MERN E-Commerce Web Application developed using React, Node.js, Express.js, MongoDB, and Vite. It allows users to browse products, add items to cart and wishlist, place orders, and view order history.

---

## 🚀 Features

- User Registration & Login
- Product Listing
- Product Details
- Add to Cart
- Wishlist
- Checkout
- Order Placement
- My Orders
- Delivery Date
- Cash on Delivery
- Admin Product Management
- Responsive UI

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/bhrama123/Neocart.git
```

### Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd server
npm install
```

### Run Backend

```bash
npm start
```

### Run Frontend

```bash
npm run dev
```
